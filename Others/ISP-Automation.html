Java Application and Database Project

ISP Automation

A desktop application that handles data for a local ISP. It handles all data including its customer's details, billing system, customer fees, employees details, employees salaries and other details. It detects defaulter customers and automatically who are not paying their dues make a list of them to display to admin whenever needs. All you have to do is save your info and let the application handles everything.

Front end for this particular project was designed in Java Swing. Each scene and frame was designed using Java Swing libraries and all backend functionalities was done in OOP (Object Oriented Programming) using Java.

A database was designed and connected to this application. Database was proactive and have a backup for each and every change at any point. Database schema was designed in normalized form and was created in SQL Server query language. Here are some sample images for final look of application.
